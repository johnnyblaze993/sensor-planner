# Practice exercises

Work in a temporary branch. Each language folder contains a starter problem.
Try the stated tests before reading `solutions/solution-notes.md`. The solutions
are guidance, not the only acceptable implementation.

Suggested loop:

1. Predict what should happen.
2. Write or update a test.
3. Observe the failure.
4. Make the smallest implementation change.
5. Run all tests.
6. Explain the change aloud or in notes.
